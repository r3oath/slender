    Slender CMS

    A lightweight, flexible and utterly simple content management system plugin for Laravel.

    Why use Slender?

    Slender can turn a static website into a rich and dynamic one. Simply tagging where you want content to be shown, you'll be able to change what is displayed there at any time.

    Tagging is as simple as {{ Slender::html('Home: Main Content', 'Some Description') }}

    What does Slender currently support?

    HTML, Text, CSS, JS (JavaScript, jQuery etc) and Images. All of which are managed from the dashboard.
